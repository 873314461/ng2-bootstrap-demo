# Angular2Bootstrap
此项目用来示范Angular2结合Bootstrap的用法。

## 用法

	npm install
	npm install -g angular-cli
	ng serve

打开你的浏览器，输入http://localhost:4200/

## ng2-bootstrap官方的仓库

https://github.com/valor-software/ng2-bootstrap
